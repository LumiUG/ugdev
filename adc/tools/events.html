Dialog Settings

Dialog indexes start at 0, whenever you want an event to trigger, keep that in mind.
Emptying all dialogs can make a good use for invisible triggers even if there's no dialog,
you can set a trigger to 0, and it'll happen without showing anything to the player.

Text speed is customizable, but it's recommended to leave it at its default.
CanBeSkipped variable controls whether the player can "spam" through text.
